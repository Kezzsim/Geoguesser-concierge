# GeoGuesser Concierge 🌎

## A party game with a furry twist

[PLAY THE GAME HERE](https://kezzsim.github.io/Geoguesser-concierge/)

Designed for @kezzsim's pannel at FWA 2026, following *Competitive Ikea Assembly* and other notable panel games.

- Previous panel [Oh the Furbanity!](https://github.com/Kezzsim/oh-the-furbanity) at FWA 202ß

### Game Master View

- Open `manifest.csv` to get an idea about available sessions and answers
- Load that view on laptop for audience use

### Player view

- Enter scenario from the Game Master
- Look around the photo or photosphere for 5 seconds
- If it was a photosphere, place a point on the map to be scored
- Enter what you think the furry convention (or other fandom reference) is called
- Answer multiple choice questions about the location you've recieved
- Get your final score and show it to the game master

### Local Development

Run a server with:
`python3 -m http.server 8080`
